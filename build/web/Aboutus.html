About Mule Banking System

      The goal of the Mule Banking System is to provide electronic system which enables user to do all banking activities through online using internet. A Mule Banking System will help users to:

  • Creating an online account.
  • Subscribe for alerts through emails and messages.
  • Process checking and saving accounts.
  • Paying bills through online.
  • View and Request for transaction statements.
